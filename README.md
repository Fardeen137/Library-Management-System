# Library Management System 📚
A simple C++ program to manage a library, allowing users to add, borrow, and return books.

## 🚀 Features
- Add books 📖
- Search books 🔍
- Borrow and return books ✅
- Remove books ❌
- List all books 📋

## 🔧 How to Compile & Run
```bash
g++ -o library library.cpp
./library
